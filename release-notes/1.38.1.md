## Changes

- Fixed a black-screen render crash that can be caused when a cyclic power has text in the "trigger" field.
- Fixed a render bug for the skill check dialog that prevented its radio options from displaying correctly.
- Added `game.archmage.wrapRolls()` method to enrich text with inline rolls in arbitrary places like macros and journal entries.
